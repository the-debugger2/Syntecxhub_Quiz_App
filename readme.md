# Quiz App

A simple interactive **Quiz Application** built using **HTML, CSS, and JavaScript**. The app presents multiple-choice questions one at a time, tracks the user's score, and displays the correct answers after the quiz is completed.

## Features
- Multiple choice questions
- One question displayed at a time
- Score tracking
- Final results display
- Shows correct answers after completion
- Restart quiz option
- Clean and simple UI

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How It Works
1. Questions are stored in a JavaScript array.
2. The app displays one question at a time.
3. User selects an answer.
4. The score updates if the answer is correct.
5. After the last question, the final score and correct answers are shown.
6. Users can restart the quiz.
